## 物理约束

**算挂点和 BS 只能在楼上做**，因为要拉 main 分支。

## 上下环境差异

| | 楼上 | 楼下 |
|---|---|---|
| 拉 main | ✅ | ❌ |
| XL 资源 | 有 | 没有 |
| 引擎 | 一种 | 另一种（不同） |

## 历史

**以前**：陈川、小石可以在下面拉 main → 可以在楼下做挂点
**现在**：不行 → 外派挂点工作受限

## 影响

- 挂点 / BS 相关工作**必须调度到楼上时段**
- 外派如果只能楼下，接不了挂点单
- 需要**跨楼层协调时间**

## How to apply

- 派挂点任务前先问：**这个人这周能上楼几次**？
- 攒批：一次上楼把好几件挂点都算掉，别一次一趟
- 长期看：**能不能让下面也能拉 main**？如果不行，把限制成本算进项目排期（挂点 = 楼上时间成本）
