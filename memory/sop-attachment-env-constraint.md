## 约束 1：楼下机房无外网 / 权限不全

**楼下机房**（可能是引擎组所在）：
- **无外网**
- **权限不全**（引擎相关素材/资料缺失）
- **引擎组查资料需要天蓝代为查找**（这是天蓝时间被吃掉的一个来源）

**楼上美术区**：有外网 + 权限全

## 约束 2：内部 vs 外派 权限有区分

**权限差异不只区分楼上楼下工位，也区分内部员工与外派人员**：
- 外派权限比内部员工少
- 有些资源/工具外派不能直接访问 → 天蓝要帮忙拉数据 / 转文件

## 约束 3：挂点/BS 只能楼上做

- **拉 main 分支必须在楼上**
- 以前陈川、小石可以下面拉 main，现在**不行**了
- 所以外派挂点工作受限
- 派挂点任务前必须问：**这个人这周能上楼几次？**

## 影响

- 天蓝**日常有一大块时间**在做"跨环境代查"（帮引擎组查东西、帮外派拉数据）
- 这部分时间是**看不见的**，容易被忽略 → 应该量化（见 sop-work-time-tracking）
- 派任务时**要把"跨楼层时间成本"算进去**
