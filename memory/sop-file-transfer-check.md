## 问题

**出现过的漏传案例**：
- 像素耳环 —— 只传了材质，**模型没传**
- 橘子 —— 直接没传

**"做了但没传"** 是常见遗漏，缺乏系统检查机制。

**Why:** 提交时精力都在"做完这个"上，很少回头对"传了没"。做了 vs 传了是两件事。

## How to apply

### 短期（个人 checklist）

**提交任何一个配饰时，走 3 步核对**：

1. **模型**（.fbx / .max 等主文件）
2. **贴图**（.tga / .png 等，含各版本）
3. **材质** / **附件**（材质球、渲染配置、参考图等）

三样**同时到位**才算真的提交。缺任何一样 → 挂在群里 "@我 补"，别默认对方会主动追。

### 中期（自动化）

**跟配饰主表联动**：
- 主表标"提交状态"字段（已交 / 缺 X / 缺 Y）
- 定期扫：**列出 "已交但 X/Y 缺"** 的配饰 → 提醒补齐

### 长期（流程）

- 交付时按**打包模板**走（zip 含固定的三个文件夹：model / texture / material）
- 缺任何一个文件夹为空 → 提交失败，别让残缺文件流到下一环节
