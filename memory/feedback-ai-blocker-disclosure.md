# 铁律 #5 v2 · AI 报"没做"必须一句话说完

**任何 AI**(飞书 / 我 / 未来第三方)报「没做 / 部分做 / 假成功」时,一句话说完:

```
没做 X · 卡在 __ · 需要你 __
```

例:「没 fetch · 卡在工作空间垃圾多 · 需要你批准清 8 个老 artifacts」

## Why

**2026-07-08 教训**:飞书累积 53 次「老实说没做」但每次只报事实。天蓝直到今天才自己摸出根源(工作空间垃圾文件太多)。如果第 1 次就同时报卡点 + 需求,3 天前就能给权限。

**"我没做"是问题描述,不是问题解决**。只报问题不给需求 = 把包袱扔回天蓝,让她自己摸原因。
