# 收银员视角完整剧本（分镜式）

**人设基底**：24 岁，话少，分寸感强，习惯用沉默和小动作藏情绪；主动申请通宵班，比起空荡的出租屋，更习惯和陌生人的安静共处。

**叙事节奏**：默认行为是基底，玩家选择改变动作细节，不改变人物底色；所有心理活动仅玩家可见，角色间永不挑明。

**叙事人称**：剧本用第三人称描述便于阅读，实际游戏文本用**第二人称"你"**（视觉小说惯例）。

## 【开场 01:47 静帧】

**场景**：冷调嵌入式柔光铺满店面，关东煮锅浮着一层细白热气，落地窗蒙着薄雾，门外街灯昏黄，风卷着落叶滚过台阶。

**店员动作**：
斜靠在收银台边缘，右耳塞着半只有线耳机，音量压得极低，只剩模糊的旋律漏出来。左手无意识转着一支黑色马克笔，目光落在玻璃门外的空街上，没什么焦点。
收银台底下的手机震了两下，短促的两声。他垂眼扫了一眼。

**手机 overlay 内容（打开后 UI 里显示）**：
- 底部：黑色锁屏 + 极简音乐播放器条 —— 模糊专辑封面 + 播放 / 切歌按钮（对应他戴着半只耳机在听歌的设定）
- 中间：横幅通知弹出「**运营商提醒 · 您的账户余额已不足 10 元**」，点亮 3 秒后自动暗下去
- 三档选择叠在下方（承接现有 phone beat 的 earbud 逻辑：half / one / off）

指尖碰了碰锁屏，没解锁，随手倒扣了回去。
耳机里的歌刚好切到副歌，他指尖跟着台沿轻轻点了两下，又停住。

**玩家可操作**：全场景开放自由点击
- 点收银台：看到他倒扣的手机、半盒没拆的薄荷糖、整齐码放的硬币
- 点门口台阶：橘猫扒着空水碗小声叫，猫碗歪在一边
- 点货架区、窗边吧台：空无一人，只有整齐的商品和空座位

**推进触发**：玩家点击「收银台」，门铃叮的一声轻响，第一位客人进店。

## 【第一阶段 01:50-01:57 独居老人】

### 进店瞬间
门铃响的同时，他抬手摘下耳机，线绕两圈揣进卫衣口袋，直起身。目光扫过门口，看见老人佝偻着身子拐进来，径直走向左侧货架区的热奶柜。
他没出声招呼，低下头继续整理零钱盒，指尖把一元硬币按面额码得整整齐齐。

### 可触发碎片 · 货架区（玩家点击货架触发）
视线越过收银台，能看见老人踮着脚，伸手够上层最里面的那盒牛奶。他晃了一下，扶住柜边稳住身子，又接着够。

**三档选择对应店员动作**：
- 冷漠档：目光扫过就收回，继续低头数硬币，像没看见
- 默认档：抬眼盯了两秒，右手撑了下收银台边缘，身体微微前倾；见老人自己够到了，又默默坐回去
- 热心档：没出声，绕出收银台走过去，伸手取下那盒牛奶递到老人手里，转身就往回走，全程没说话

老人拿到牛奶后，站在原地慢慢摩挲了两下包装盒，又慢悠悠地在货架前晃了晃，才往收银台方向挪。

### 结账锚点（玩家再次点击收银台触发）
老人挪到台前，把布零钱包放在台面上，拉链拉了两下才拉开。他掏出一把硬币，手抖着数了两遍，没数清。

> 老人："小伙子……麻烦结下账。"

**三档选择对应店员反应**：
- 冷漠档："可以扫二维码，更快。" 语气平淡，指尖敲了敲收款码牌。老人愣了愣，皱着眉继续数硬币，他才不耐烦地伸手接过硬币，数完找零推过去，全程没抬头
- 默认档：等老人数了第三下还没数清，伸手把硬币接过来，指尖飞快点数，找零按面额码好推过去，没说话
- 热心档：先报了总价，伸手把散硬币拢到自己跟前，一枚枚理整齐数完；找零的纸币叠得方正，硬币码成小摞递过去，"您拿好"

老人揣好零钱，拎着牛奶慢慢走出去。玻璃门合上的瞬间，店里又只剩关东煮的咕嘟声。

## 【第二阶段 02:00-02:05 夜班护士 + 保洁阿姨】

### 空窗期默认状态
老人走后，他靠回椅背上，没戴耳机，指尖转着笔，目光落在关东煮锅升腾的热气上。
玩家可自由切换场景，触发不同碎片：
- 点收银台 / 热食区：触发「煮关东煮」选择
- 点窗边吧台：空座位，杯垫摆得整齐
- 点门口：橘猫蜷在台阶上避风

### 护士进店（玩家点击收银台触发）
门铃再响。穿米白外套的女生走进来，身上带着淡淡的消毒水味，头发有点乱，眼底泛着红血丝。
他抬眼和她对视了半秒，微微点了下头，没说话——是常来的夜班护士，不用多招呼。
护士左拐进货架区，拿了一盒热牛奶，又拐向右侧窗边吧台，选了靠里的座位坐下，把外套搭在椅背上，低头揉着手腕。

### 可触发碎片
1. **货架区交叉彩蛋**：如果上一阶段老人还没结账，护士途经热奶柜时会撞见老人够牛奶，触发「护士帮忙拿牛奶」的互动；店员站在收银台看着，不插手
2. **窗边吧台**：玩家点过去，能看见护士袖口露出的碘伏印，她按着手腕，动作很轻，像在忍疼
3. **热食区**：玩家选择煮关东煮，店员起身拿串下锅，热气漫上来，窗边的护士往这边看了一眼，又收回目光

### 保洁阿姨进店（小锚点，护士歇脚期间触发）
玩家再次点击收银台，保洁阿姨推门进来，穿着橙黄色马甲，手里攥着布袋子，直奔临期食品架。
两分钟后，阿姨拿着两袋面包过来结账。

> 保洁阿姨："小伙子，这临期的是买一送一吧？"

**三档选择**：
- 冷漠档："嗯。" 低头扫码，递过去
- 默认档："对，我再给你拿一袋。" 转身从货架补了一袋递过去
- 热心档：扫码后顺手从货架拿了一枚独立包装的小面包，轻轻放进袋子里。（声音很轻）"搭着吃。"
  - 依据：对齐店员"沉默细心、不多话"的统一人设，跟给司机递温水 / 给少年递充电线的"做了不说"风格一致。

保洁走后，护士还坐在窗边歇脚，店里很安静。

### 护士结账锚点（玩家再次点击收银台触发）
护士起身走过来，手里拿着牛奶，另一只手拿着两串关东煮（如果煮了的话），手机递到扫码区。

> 护士："你好，结账。"

**三档选择**：
- 冷漠档：低头扫完码，把东西推过去，全程没抬头
- 默认档：扫到她袖口的碘伏印，手上没停，扫完淡淡问了句"袋子要吗"
- 热心档：扫完码顺手拿了厚袋子，把关东煮单独套了一层，"汤容易洒，拿好"

护士拎着东西出门，玻璃门晃了晃。他看着她的背影消失在路口，收回目光。

## 【第三阶段 02:08-02:10 网约车司机】

### 空窗期默认状态
他拿起抹布擦了擦收银台台面，又把关东煮的浮沫撇了撇。窗外的风好像大了点，落地窗轻轻响。

### 司机进店（玩家点击收银台触发）
男人推门进来，身上带着夜风寒气，肩膀绷得很紧，走路时腰微微僵着。他没往里面走，直接坐到窗边吧台最靠门的位置，背对着收银台，抬手反复按后腰。
他扫了一眼，没说话，低下头整理货单。

### 可触发碎片 · 窗边吧台（玩家点击窗边触发）
能看见男人侧着脸，眉头皱着，手掌按在腰上，按一下缓两秒，像老毛病了。

**三档选择**：
- 冷漠档：看了一眼就收回目光，继续理货单
- 默认档：伸手按了下温控面板，把窗边的暖气调高了半格，没起身，也没出声
- 热心档：起身假装去整理旁边的冷饮冰箱，顺手把墙角闲置的靠垫，悄悄挪到他座位旁边的台面上，转身就走

男人坐了八分钟左右，直起身活动了一下腰，往收银台走。

### 结账锚点（玩家点击收银台触发）
男人拿了一瓶常温白水，放在台面上，指尖冻得发红。

> 司机："拿瓶水。"

**三档选择**：
- 冷漠档：扫码收钱，把水推过去，零交流
- 默认档：扫到他冻红的指尖，停顿了半秒，没说话，把水往他那边推了推
- 热心档：扫完码，转身从保温机接了一杯温白水递过去，"凉的伤胃，这个免费"

司机愣了一下，小声道了谢，攥着水杯出门。

## 【第四阶段 02:13-02:20 代驾小哥 + 叛逆少年】

### 空窗期默认状态
他喝了一口员工杯里的温水，看向窗外，街上车更少了。

### 代驾小哥进店（玩家点击收银台触发）
穿头盔的小哥快步走进来，头盔挂在胳膊上，直奔冷藏柜拿了瓶矿泉水，又抓了两个茶叶蛋。

> 代驾小哥："麻烦快点结，我那边还有单等着。"

**三档选择**：
- 冷漠档：低头快速扫码，递过去，没说话
- 默认档："袋子要吗？" 手上动作没停
- 热心档："茶叶蛋给你拿的热的。" 顺手塞了个纸巾在袋子里

小哥道了声谢，快步跑出去，电动车引擎声远去。

### 少年进店（玩家再次点击收银台触发）
门铃轻响，穿黑色卫衣的少年低着头进来，刘海遮住眼睛。他没往店里走，径直拐去门口台阶，背着帆布包蹲下来，从包里掏出分装猫粮。
他抬眼扫了一下门口，没出声，继续擦台面。

### 可触发碎片 · 门口台阶（玩家点击门口触发）
少年蹲在台阶上，橘猫蹭他的手背，他动作很轻，一下一下顺着猫毛。他肩膀很瘦，卫衣袖子有点长，盖住半只手。
此时猫碗的状态由玩家之前的选择决定：空的 / 有水 / 有粮有水，少年会对应做出不同的小动作。

### 少年结账锚点（玩家点击收银台触发）
少年低着头走进来，手里攥着一袋最便宜的猫粮，放在台面上。掏手机时按了好几下，屏幕黑着，他手指顿了顿，更用力地按了一下，还是没亮。

> 少年："……这个。" 声音很小，几乎听不清。

**三档选择**：
- 冷漠档：扫完码把猫粮推过去，看见黑屏也当没看见，全程没抬头
- 默认档：抬眼扫了一下黑屏的手机，没说话，继续低头整理零钱
- 热心档：弯腰从柜台底下摸出一根备用充电线，推到他面前，没说话，也没等他道谢

少年攥着充电线，小声说了句谢谢，头埋得更低了。

## 【收尾 02:25 深夜过半】

少年结完账，出门蹲回台阶上，背对着店门，抱着膝盖。橘猫蜷在他脚边。
他靠回收银台椅背上，没戴耳机，店里安静得能听见冷柜的低噪。玻璃门上映着他模糊的影子，门外的少年缩成小小的一团。

### 可触发最终碎片 · 留门缝（玩家点击门口触发）

**三档选择**：
- 冷漠档：看了一眼，收回目光，继续坐着发呆
- 默认档：偶尔抬眼扫一下门口，怕他出事，但没开门
- 热心档：起身走到门边，把玻璃门拉开一条细缝，暖气顺着缝漏出去。他心里找了个借口：店里闷，透透气

### 最终静帧（结局单视角收束 · 只写店员这一晚）
他重新坐回收银台，拿起那半只耳机，塞回耳朵里，没开声音。
关东煮还在咕嘟。台阶上的少年还蹲着。耳机里的歌循环到第三遍。
玻璃窗上映着他自己模糊的影子，比他以为的样子更疲惫一点。

这个夜晚还有很长。

### 结局 flag 增量变体（每个 flag 1-2 句 · 插入在"这个夜晚还有很长"之前）

**规则**：每个 flag 命中暖档时，追加 1-2 句静帧细节（不改核心结构、不做好坏区分）。多个 flag 同时命中就多几行——玩家自己回味这一晚做了些什么。

| flag 命中值 | 追加静帧句 |
|---|---|
| `w.catBowl === 'water'` | 门外那只碗歪得没那么厉害了。 |
| `w.catBowl === 'full'` | 台阶上有几粒散落的猫粮。 |
| `w.milkHelp === 'ready'` | 收银台边缘的木漆磨得比别处深一点。 |
| `w.milkHelp === 'given'` | 热奶柜顶层空出的位置很整齐。 |
| `w.odenState === 'cold'` | 关东煮锅里剩着几根空签子。 |
| `w.odenState === 'hot'` | 保温档亮着。他没关。 |
| `w.cashService === 'fast'` | 收银机抽屉里的硬币按面额码成三堆。 |
| `w.seatHeater === 'normal'` | 窗那边不再冷。 |
| `w.seatHeater === 'cushion'` | 靠垫落回了原来的位置。 |
| `w.nurseCheckout === 'warm'` | 收银台上多出一小张纸巾。是他刚才垫热奶盒的。 |
| `w.driverCheckout === 'warm'` | 保温机的水位下去了小半格。 |
| `w.cleanerCheckout === 'warm'` | 台面上少了一个薄塑料袋。 |
| `w.driverAltCheckout === 'warm'` | 门外的电动车已经没影了。 |
| `w.chargerState === 'given'` | 柜台底下那根充电线还没收。 |
| `w.doorGap === 'watched'` | 收银台后面的角度，正好能看到台阶上那团缩着的影子。 |
| `w.doorGap === 'open'` | 门缝漏出的暖光落在台阶上，那团影子挪近了一点点。 |
| `w.nurseObserved === true` | 窗边高脚凳上还留着一点体温的印子。 |
| `w.teenObserved === true` | 玻璃窗上映着的不只是他一个人。 |

**呈现规则**：
- 按 **CASHIER_TIMELINE 的 step 固定顺序**排列（老人 → 护士 → 保洁 → 司机 → 代驾 → 少年 → 收尾），**不随玩家实际触发顺序变动**。保证结局叙事节奏稳定，从入夜到深夜逐步铺开细节，避免玩家乱序探索导致语序混乱
- 每句独占一行 · 段间距同正文
- 全部追加到"耳机里的歌循环到第三遍。/ 玻璃窗上映着他自己模糊的影子..."之后、"这个夜晚还有很长。"之前
- 玩家触发顺序只决定"有没有这句细节"，不决定"这句话放在哪个位置"
- **未命中任何暖档** → 不追加任何句 → 直接跳到"这个夜晚还有很长"（也是一种结局，静帧感更强）

### 结局画面的场景视觉呼应（跟文本变体并行 · v1.0 落地）

**规则**：结局静帧画面里的场景物件，也跟着 flag 命中变化——**不做任何提示**，玩家发现是惊喜，没发现完全不影响。呼应"无人知晓的温柔"内核。

| flag 命中值 | 场景视觉呼应（改结局画面 CSS / SVG 图层可见性） |
|---|---|
