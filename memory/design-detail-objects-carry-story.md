在《凌晨两点的便利店》里，画面里的细节物件应当**承载角色的故事**，而不是纯粹装饰。用户明确表示喜欢这种设计。

**Why:** 游戏的核心气质是**克制**——不叫喊、不解释、不"教育"。角色不承认自己的善意，也不承认自己的疲惫。物件是唯一能替他们说话又不显山露水的媒介。用户："可以通过一些这样的细节 体现角色的故事细节"。

**How to apply:**

- 每个角色的 POV 场景（尤其是店员的柜台内视角 register-inside、护士 / 司机 / 少年将来的第一人称场景），至少放一个能"漏出"角色内心的**手写物件 / 私人物件**：一个杯子上的字、一张便签、一个护身符、一片贴纸、一句锁屏、一支烟盒上写的字……
- 优先选**手写体 / 印刷体旁边的手写涂鸦**——传递"随身、私人、无意间流露"的感觉。
- 内容要**贴合角色的具体人设背景**，不要空泛。**人设正典见 [[midnight-store-cashier-canon]]**——里面对店员、护士、司机、少年、老人的基底描述是唯一权威。
- 生成图时把这类物件**写进 prompt**，让 AI 主动构图放进去。
- 别塞太多——一个场景 1-2 个"会说话的物件"就够，太多反而喧宾夺主。

**旧例子已作废**：之前的 memory 里举过"店员杯子上写「且慢且静」/ 便签「记得吃药 - 妈」/ 锁屏「还好」"这些例子——都对应**旧基底"店员等妈化疗"**。新基底是**"24 岁、话少、独居、主动申请通宵班、比起空出租屋更习惯陌生人的安静共处"**（见 [[midnight-store-cashier-canon]]）。旧例子跟新基底冲突，都不要再套用。

看到用户反馈"这个细节好"时，把它记住，之后每次做类似图（角色 POV / 心境场景）都要主动嵌一个——但先确认物件跟当前正典人设一致再放进图里。
