## 创作初衷（重要 · 影响所有内容决策）

**现实经历较多糟心事** → 想以便利店为舞台做**群像叙事** → 舒缓负面情绪。

## 现存卡点

**平时很少细致观察人群** → **缺少人物细节素材** → **不清楚如何继续推进剧情**

## How to apply（写文本/内容时）

- 内容基调应该是**温和 / 治愈**，不是黑暗
- 群像 = **多个陌生人的短暂交集**，每个人有独立小故事
- **具体人物细节**是关键（不是脸谱化，要有"这个人这一刻的真实感"）
- 剧情推进的**素材来源**是问题 → 需要主动做"人物观察积累"（观察路人 / 记录同事对话细节 / 电视剧片段），不能坐等灵感
