## pin 一下

**提交完成截图或文件时**，用 pin 功能标记，方便后续查找。

## 打包 vs 沟通

- **整理文件时** → 提交**完整包**（高模 + 低模 + Unity 效果截图）
- **过程沟通时** → 只提交**需要的文件**（不要每次都塞整包）

## 引擎截图会压缩，正常现象

引擎内截图会有**压缩导致的朦胧感**，这是正常的。**以引擎内实际显示效果为准**，不必追求截图清晰度。

## 归档版本优先级

配饰模型截图归档到 Base 表"模型截图"字段时，**按以下优先级取最新非空**：

```
V04 → V03 → V02 → V01 → 模型截图
```

写入时**覆盖旧值**（这里 OK 覆盖，因为我们只是取更新的版本；跟 feedback-only-fill-blanks 的"不覆盖已有数据"是不同场景）。
