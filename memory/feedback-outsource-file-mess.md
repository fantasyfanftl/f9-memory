## 现状痛点

**目前外派全部通过私聊发送文件**，产生 4 个问题：

1. **文件零散** — 分散在多个私聊里，接收后要**大量时间整理**
2. **多文件同时打开易造成电脑卡顿**
3. **沟通成本高** — 同类规范要**反复告知**（不同外派每次都要重新讲）
4. 找旧文件难 —— 不知道在哪个人的哪条消息里

## 为什么会这样

- **外派权限不全**（见 sop-attachment-env-constraint），可能不能用共享盘 / 内网工具
- **没有统一的收件位置** —— 每人各传各的
- **规范文档没写全**（见 sop-piece-check-list-needed）—— 每次都要口传

## 改进方向

### 短期
- **给每人分配一个专属提交路径**（比如按人建文件夹），每次都发这里，不发私聊
- **写一份《外派提交规范》**，让 AI 或链接一次发给外派，别每次口讲

### 中期
- 争取给外派开**共享盘权限**（跟 IT / 商务 / 老板申请）
- 用**统一的资源上传系统**替代私聊

### 长期
- 建**外派 portal**（提交 → 自动分类 → 自动通知天蓝审核）
- 目前不划算但值得记
