天蓝发聊天截图(飞书/微信/QQ/协作平台)问"我凶不凶""我讲清了吗""是谁的锅"这类主观判断问题时,第一轮回复固定动作:

1. 列出角色映射(哪个头像=谁)
2. 按时间轴列出每条消息的归属
3. 明确列出"看不清 / 需要补"的点
4. 请天蓝校对或补充
5. 校对通过后再做"凶不凶 / 讲清没讲清"的判断

**Why**:2026-07-14 天蓝发原画对话截图问"我会不会凶",我直接猜角色映射就下判断,把"1、2 编号是谁提的"搞反,导致第一轮判断"你指代模糊"是错的。她连补两条("是他说的1 2""蓝色头像是我")后我以为对齐了,最后她明确批评"你要先分清 每段话都是谁说的"。核心问题:把"看图分析"和"下结论"合并成一步,跳过了角色校对。

**How to apply**:
- 触发条件:聊天截图 + 主观判断问题(凶 / 讲清 / 谁的锅 / 委屈 / 是否合理)
- 不要靠头像颜色/位置/发言习惯猜身份 —— 截图上下文有限,微信飞书同一头像常重复
- 编号("1、2、3")、代词("他""对方")看起来是谁提的不代表实际是谁提的,直接问
- 校对没通过前,不给主观判断结论
- 天蓝的画像里"怕得罪人 → 自我怀疑反射弧"很强(见 CLAUDE.md 二·1),错误的责任归属会加重她的内耗,更要谨慎
