lycheeGame 项目每次改完 HTML 后我会打开浏览器让用户看效果。之前我习惯提示「Ctrl+Shift+R 强刷,别吃缓存」。用户明确要求换掉这个啰嗦的快捷键写法。

**Why:** 每次都重复完整快捷键太啰嗦,用户已经知道怎么强刷。

**How to apply:**
- 就说「强刷一下」或「刷新看看」,不要拼 `Ctrl+Shift+R`
- 也不要提 F5 / 右键刷新 之类的具体操作,直接省略
- 如果真需要提醒(比如新加了图片资源、缓存问题),简单说「强刷」两字够了
- 相关:[[feedback-open-after-update]] 依然要打开浏览器展示效果
