## 观察维度

| 维度 | 具体能力 |
|---|---|
| **拆分** | 1 条输入拆成几条待办 · 拆得对吗 |
| **类别匹配** | 内容 → 类别 (🎨🔗📊🎁🧵📞🤖🏠) 判断准吗 |
| **动作方向** | 主语识别 (谁给谁/谁做/谁催) · 会不会反向 |
| **位置判断** | 工作类默认飞书表 · AI 协作默认 GitHub · 会不会问错 |
| **优先级** | 遵守"默认🔴今日"规则吗 |
| **阻塞点 vs 完成日** | DL 该放完成日不是阻塞点 · 会混吗 |
| **主动申报缺失** | 缺主语/优先级/位置时 · 会主动问吗 (还是硬猜) |
