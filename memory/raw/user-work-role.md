## 完整配饰工作流

详见 [project-accessory-workflow-fullcycle](project-accessory-workflow-fullcycle.md)：

```
运营（需求）→ 原画 PM（分配编号）→ 至尊菜刀等原画（配饰设定稿）→ 天蓝
  ↓
天蓝判断制作方案：内部自制 or 外派发包
  ↓ 若外派：对接商务确认报价、排期；同步对接外派人员统一规范、整理原画资料
  ↓
3D 配饰制作完成 → 效果图给原画审核
  ↓ 原画通过 → 规整文件上传统一资源路径
  ↓
绑定组做骨骼绑定 → 资产组合入资产
  ↓
天蓝做配饰挂点 → 原画确认挂点效果 → 全流程结束
```

## 日常协作小组

- **运营** — 提投放需求
- **原画组** — 设定稿输出、审核
- **商务** — 外包报价、排期对接
- **外派 / 外包** — 制作 hands-on
- **绑定组** — 骨骼绑定
- **资产组** — 资产合入
- **解算组** — 特效解算
- **特效组** — 特效制作

**日常无交集**：美术总监 / QA lead 等（后续有交集再补）
