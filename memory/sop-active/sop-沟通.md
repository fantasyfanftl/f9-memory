## 沟通优先级 4 层

### 🔴 最高紧急(立刻放下秒回)
- 渠道: 飞书 IM · 飞书电话
- 人: 乘凉 · 麦大麦 · PM 牛二 · 至尊菜刀
- 场景: 版本阻塞 · 管线报错 · 紧急线上问题

### 🟠 高优先(30 min 内回)
- 渠道: 飞书群 · 企业微信
- 人: 主策 · QA lead · 同组成员 · 外包对接
- 场景: 当期验收 · 需求变更 · bug 核对

### 🟡 常规(2h 内回 · 完成手头模块再处理)
- 渠道: 飞书文档评论 · 邮件
- 人: 运营 · 宣发 · 场景/动画同事 · 行政
- 场景: 活动素材 · 渲染需求 · 流程审批

### 🟢 低优先(当日下班前统一回)
- 渠道: 私人微信
- 人: 副业合作方 · 外部供应商 · 闲聊同事

**硬性规则**:
- 乘凉: 任何渠道立即回(唯一特权)

## 任务分级(处理打断)

### 4 类必接

| 类型 | 特征 | 应对 |
|---|---|---|
| A 教学式 | 技术连环问(陈明星/王智超) | 立回"半小时后 · 先看 sop-XX" · 答完顺手沉淀 |
| B 咨询式 | 组长/PM 要决定(麦大麦) | 立即回 · 不确定说"15 min 后回" |
| C 提醒式 | 内部"XX 别漏" | 秒回"收到" · 加 today's list |
| D 转发式 | 你做 middleman | 立即转 + educate 外派下次直接对接 · 累计 3-5 次推直连 |

### 2 类可推

- **求助式**(跨组非你岗) · "我不熟找 X" / "半小时后回" · 老板/菜刀例外必接
- **闲聊** · 下班前批量回

### 判断树

```
消息 → 看谁 + 看第一句
├ 老板/菜刀/麦大麦/牛二 → 立即回
├ 外派技术问题 → A 教学式(重复问题写 SOP)
├ 组长要决定 → B 咨询式
├ 内部提醒 → C 提醒式
├ 别组求助 → 判断岗位 → 求助式
└ 无关 → 批量
```

### 回复模板

- 教学: 「先看 sop-XX · 看完再来」
- 咨询: 「我 15 min 后回」
- 提醒: 「收到 · 做完这段处理」
- 求助: 「我不熟 · 你找 X 更快」/「半小时后回」

## 跨部门需求接入

**别人来提配饰需求时 · 你怎么接**

### 必要信息(缺任一 · 不接 / 让他先补)

- 原画 / 参考图
- 类型(耳饰/帽子/项链/面饰等)
- 交付时间
- 引擎用途(什么场景要用)
- 项目编号(牛二给的)

### 什么时候必接 vs 可拒

| 来源 | 接不接 |
|---|---|
| 老板 / 菜刀 | 必接 · 任何渠道立即回 |
| 麦大麦 / 牛二 | 必接 · 走排期评估 |
| 主策 · QA lead | 走排期评估 |
| 别组随口 · 非配饰岗 | 可拒 or 延后 |
| 信息不全 | 拒 · 让他先补 |

### 排期反馈时效

- 收到需求 → **24h 内给排期反馈**(能不能接 · 什么时候交)
- 无法在 24h 内评估 → 明说"我 X 天给你答复"

### 回复模板

- 信息不全 · 拒: 「你先给我原画 + 类型 + 交付时间 · 我再评估」
- 排期评估: 「收到 · 我 X 天内给你排期」
- 走排期后可接: 「排上了 · 预计 X 月 X 日交付」
- 走排期后不接: 「本周排满 · 建议下周 or 找 X 顶下」

## 抽象反馈处理法

**场景** · 对方给"更磨砂一点"·"感觉不对"·"再调整下"等**模糊反馈** · 或直接说"好废话"

### 3 步骤

**1 · 对方给抽象反馈 → 拆成 3 个具体方向让对方选**
- 不追问"到底怎么改"(对方也没想清楚)
- 主动列 2-3 个具体路径 · 让对方选一个
- 例:"更磨砂粗糙一点?颗粒感明显?还是别的方向?"

**2 · 对方仍模糊 / 补充新信息 → 明确告知"消化中"**
- 不装懂 · 不马上给方案
- 直说"(疯狂消化中)" 或 "我先想一下"
- 给自己缓冲 · 也给对方信号 · 让对方补更多信息

**3 · 给方案时 → 具体到画面 + 标注参考 + 收尾话术**
- 具体描述(而非抽象)· 例:"暗一点 + 金属质感细腻 + 磨砂部分和拉丝铝拉开区分"
- 附标注截图 · 指到具体位置(参考图 + 黄色标记)
- 收尾三件套:
  - 温和不硬压 · "行我们再尝试一下"
  - 提优先级 · "这个也蛮急的"
  - 说成本 · "后面都在等资源"

### 为啥这样接得住

- 抽象反馈 = 对方也没想清楚 · 你追问只会让他更没方向
- **拆 3 选 1** = 把决策权还给对方 · 但限定在你觉得可行的范围
- **消化中** = 给情绪空间 · 也给自己时间
- **具体+参考+收尾** = 给对方"照做就行"的路径 · 减少再来回

### 触发关键词

**要用这套** · 对方说"感觉不对"/"再调调"/"好废话"/"我说不清楚"/"你看着改"

**不用** · 已有明确方向的反馈(照改就是)
