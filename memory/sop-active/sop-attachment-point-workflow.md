## 全流程

```
挂点美术(石静玲/岳淑茹) · 调位置 + 截图 → "挂点疑难杂症解决"群 Pin
     ↓
天蓝确认 OK
     ↓
BS 技术(陈川) Houdini 算 BS · 先男女主默认发型
     ↓
天蓝确认默认发型效果 OK
     ↓
陈川批量算全发型
     ↓
菜刀 X3-角色资产效果核对群 回"可以了"@天蓝
     ↓
天蓝回填 Base 表
```

## 核心原则

- ❌ 不确认挂点先算 BS · ✅ 挂点先确认 · 再算 BS
- ❌ 一次算全发型 · ✅ 先默认发型 · 通过再批量
- ❌ 按脸部适配挂点 · ✅ 按发型适配
- ❌ 改一发型不检查联动 · ✅ 挂点有联动性 · 改一 → 查所有
- ❌ 帽子不逐发型检查 · ✅ 帽子必须逐发型走 · 防悬空

- 渠道: 挂点疑难杂症解决群(Pin=待办 · 完成取消)
