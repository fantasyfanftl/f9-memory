## 4 类必接

| 类型 | 特征 | 应对 |
|---|---|---|
| A 教学式 | 技术连环问(陈明星/王智超) | 立回"半小时后 · 先看 sop-XX" · 答完顺手沉淀 · 下次甩链接 |
| B 咨询式 | 组长/PM 要决定(麦大麦) | 立即回 · 不确定说"15 min 后回" |
| C 提醒式 | 内部"XX 别漏" | 秒回"收到" · 加 today's list |
| D 转发式 | 你做 middleman | 立即转 · 同时 educate 外派下次直接对接 · 累计 3-5 次推直接对接 |

## 2 类可推

- **求助式**(跨组非你岗) · "我不熟找 X" 或 "半小时后回" · 老板/菜刀发的例外 · 必接
- **闲聊** · 下班前批量回

## 判断树

```
消息 → 看谁 + 看第一句
├ 老板/菜刀/麦大麦/牛二 → 立即回
├ 外派技术问题 → A 教学式(重复问题写 SOP)
├ 组长要决定 → B 咨询式
├ 内部提醒 → C 提醒式
├ 别组求助 → 判断岗位 → 求助式
└ 无关 → 批量
```

## 回复模板

- 教学: 「先看 sop-XX · 看完再来」
- 咨询: 「我 15 min 后回」
- 提醒: 「收到 · 做完这段处理」
- 求助: 「我不熟 · 你找 X 更快」/「半小时后回」

## HARD RULES

- 乘凉 / 菜刀 · 任何渠道立即回
- 麦大麦 · 5 min 内回
- 邮件 · 2h 内回 · 不实时
- 私人微信 · 下班前统一

## 沉淀触发(每周五)

- 教学式打断几次
- 同问题问 ≥ 2 次 → 写 SOP
- 已有 SOP 外派没查 → 甩链接

## 元规则 · 别对 AI 形成打断链

拍完任务给 AI 不被打断时段(≥ 10 min) · 中间新想法先记本子 · 别打断
